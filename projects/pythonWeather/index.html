Python Weather App Project.

January 2025


Brief

The project was aimed at writing a code to get specify weather details of a town in the United Kingdom from an API (Application Programming Interface) source. The App first gets name of user, and presents a welcome message. The user then would input the name of the city, the App returns the Current Temperature, Weather Condition, Wind Speed, and Humidity of the entered city. Finally, the App expresses gratitude to the user for using the App

The App handles error of input by the user. Users are asked to retry city name entry if an invalid city was given by the user. Download a ReadMe file containing a code of the project below.
