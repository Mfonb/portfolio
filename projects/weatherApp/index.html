Python Weather App Project.

January 2025


Brief

The project was aimed at writing a code capable of getting specify weather details a towns (chosen by user) in the United Kingdom through an API (Application Programming Interface) source. The API source was the OpenWeather Map. The python built weather App first gets name of user, then welcomes to user to the App. The user is later prompted to input the name of the city of which wearther information the user seeks. The App returns the Current Temperature, Weather Condition, Wind Speed, and Humidity of the entered city. Finally, the App expresses gratitude to the user for using the App on closing.

The App handles errors of input by the user. Users are asked to retry city name entry if an invalid city is entered by the user. Download a ReadMe file containing a code of the project below.
